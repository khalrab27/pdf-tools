Welcome to PDF Tools, a web application that allows you to conveniently, efficiently, and securely split and merge PDF files.

Features:

  • Split a single PDF into multiple smaller PDFs
  • Merge mutliple PDFs into a single PDF
  • Download the resulting PDFs
  • Automatic deletion of uploaded files for greater security
  • File size limited to 64 MB to preserve space on the server as well as to prevent potential malicious activities

More features will be added soon.

Thank you for using PDF Tools!
